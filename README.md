# Stationhead Monitor

Stationhead監視システム。Cloudflare Workerが主系、Windows Local Collectorが自動フェイルオーバー系です。

- Buddiesスナップショット、コメント、キュー、曲情報
- ホストプロフィールと単独配信
- 公式ニュース連動監視
- 週間リーダーボード
- Gmail週間Recap
- D1冪等保存、取得元優先度、競合監査
